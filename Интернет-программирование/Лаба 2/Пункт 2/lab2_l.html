1. List (Список) 2. Dictionary (Словарь) 3. Function (Функция)

Список — это изменяемый упорядоченный набор элементов, который можно изменять после создания. Пример: my_list = [1, 2, 3].


Словарь — это изменяемая коллекция пар ключ-значение. Пример: my_dict = {"key": "value"}.


Функция — это блок кода, который можно повторно использовать. Функции создаются с помощью ключевого слова def. Пример: def my_function(): pass.
